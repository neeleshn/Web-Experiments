My first Bootstrap website

This page will automatically allign as we decrease the screen size because we are using container class of bootstrap css

We have used jumbotron class to provide rounded corners to the background color and the size of the text also decreases automatically if the screen size is reduced

HTML Code Snippet :
<div class="container">
<div class="jumbotron">
About the Experiment :

This experiment is about using Bootstrap "Container" Class to align the page to the middle dynamically depending upon the screen size
