JavaScript Code Snippet : 
    function buttons($scope) {
        console.log("In Controller");
        $scope.clicked = "No Button Clicked";
        $scope.button1 = function () {
            $scope.clicked = "Button1";
        };
        $scope.button2 = function () {
            $scope.clicked = "Button2";
        };
        $scope.button3 = function () {
            $scope.clicked = "Button3";
        };
        $scope.button4 = function () {
            $scope.clicked = "Button4";
        };
    }
    angular.module('myapp', []).controller('buttons100', buttons)


About the Experiment : 

This Experiment is for finding out which button is being clicked. Upon clicking of each button a function specific to that button is called by using ng-click= which displays the name of the button in the binded element.
