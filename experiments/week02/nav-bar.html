Responsive Navigation Bar

HTML Code Snippet :
<div id="maroon">
<nav><ul>
<li><a href="#">Nav1</a></li>
<li><a href="#">Nav2</a></li>
</ul></nav>
</div>


CSS Code Snippet :
@media screen and (max-width: 700px) {
#maroon {
background-color: maroon;
width:200px;
height:50%;
top : 0;
bottom : 0;
right : 0;
left : 0;
margin:left;
}
About the experiment :

In this experiment we use @media Screen to find out the screen size and design the page according to the screen size. In this experiment the navigation bar is a single horizontal line in the top of the screen for large screens like laptops and desktops. As the screen size reduces to a specific limit we align the navigation bar vertically to accomodate for smaller screens. We use CSS to design the page for each screen size. Each tag behaves differently for different screen sizes.
