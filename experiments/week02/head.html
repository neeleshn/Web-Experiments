CSS in HTML Head

HTML Code Snippet :
<head> <style> #maroon {
background-color: maroon;
width:50%;
height:500px;
position : absolute;
top : 0;
bottom : 0;
right : 0;
left : 0;
margin:auto;
}
About the experiment :

This is the same experiment as Experiment 1 but in this experiment we put all the css related stuff in the <head> tag of the html page using <style> tag to apply cascading style effects.

As the size of the page reduces the size of the image also reduces because of width:50%; in the css file.
