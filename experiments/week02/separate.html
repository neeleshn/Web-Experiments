Separate CSS file

HTML Code Snippet :
<link rel="stylesheet" type="text/css" href="../../css/week02/w02ex03.css" />


CSS Code Snippet :
#maroon {
background-color: maroon;
width:50%;
height:500px;
position : absolute;
top : 0;
bottom : 0;
right : 0;
left : 0;
margin:auto;
}
About the experiment :

This is the same experiment as Experiment 1 but in this experiment we use a separate css file to apply cascading style effects. we use <link rel="stylesheet" type="text/css"... tag to add the css file to the html page.

As the size of the page reduces the size of the image also reduces because of width:50%; in the css file.
