CSS Code Snippet :
@media screen and (max-width: 1000px) and (min-width: 650px){
img {
position:absolute;
top : 0;
bottom : 0;
right : 0;
left : 0;
margin:auto;
width:auto;
height:auto;
content:url("../../images/experiments/w02ex01.jpg");
}
About the experiment :

In this experiment we use @media Screen to find out the screen size and design the page according to the screen size. In this experiment the image and the size of the image changes as per the screen size. We have 3 variations of image for 3 kinds of screens : Desktop, Tablet and Mobile. As the screen size reduces to a specific limit we change the image and take a smaller image to fit the tablet screen and as the screen size reduces further we change the image and its size again for mobile screens. We use CSS to design the page for each screen size. Each tag behaves differently for different screen sizes.
